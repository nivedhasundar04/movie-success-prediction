# Interpretable Movie Success Prediction Using Movie Personas

## Overview
This project investigates whether segmenting movies into interpretable 
personas improves commercial success prediction, and whether success 
drivers differ across film types. We apply K-Means clustering, three 
supervised classification experiments (Logistic Regression, Random Forest, 
XGBoost), and SHAP interpretability analysis to 11,495 commercially 
released films from TMDB.

## Research Question
Does segmenting movies into interpretable personas improve the prediction 
of commercial success, and do the key drivers of success differ across 
these personas?

## Repository Structure
- `notebooks/` — Four notebooks covering the full pipeline
- `figures/` — All plots and visualizations generated by the notebooks
- `data/` — Instructions for downloading the dataset

## Dataset
This project uses the TMDB Movies Daily Updates dataset from Kaggle:
https://www.kaggle.com/datasets/alanvourch/tmdb-movies-daily-updates

Download TMDB_all_movies.csv and upload it to your Google Drive before 
running the notebooks.

## How to Reproduce Results
Run the notebooks in order in Google Colab:

1. `01_preprocessing.ipynb` — Data cleaning, feature engineering, 
   LassoCV feature selection
2. `02_clustering.ipynb` — K-Means clustering, elbow/silhouette 
   selection, PCA visualization
3. `03_modeling.ipynb` — Three experiments comparing global and 
   persona-aware models
4. `04_shap.ipynb` — SHAP interpretability analysis globally 
   and per persona
